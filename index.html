# Field Notes

A markdown editor that gets **out of the way**. Write on the
left, read on the right.

## What it does

- Live preview, no save needed
- `Cmd+B` / `Cmd+E` / `Cmd+P` to switch views
- Opens `.md`, `.markdown`, `.txt`

> Plain text outlives every app that edits it.

[Read the docs](https://example.com)
